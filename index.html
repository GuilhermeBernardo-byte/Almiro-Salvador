Descrição de projectos já feitos

Calculadora Web

A calculadora web é uma ferramenta interativa que permite realizar cálculos matemáticos de forma rápida e conveniente diretamente no navegador. Desenvolvida utilizando HTML, CSS e JavaScript, esta calculadora possui uma interface amigável e responsiva que se adapta a diferentes dispositivos.

Com a calculadora, é possível realizar operações básicas como adição, subtração, multiplicação e divisão, além de operações mais avançadas como exponenciação e uso de parênteses para definição de ordem de operações.

Lista de Contatos Web

A lista de contatos web é uma aplicação simples para gerenciar e exibir uma lista de contatos. Desenvolvida utilizando HTML, CSS e JavaScript, esta lista de contatos permite adicionar, remover e visualizar os contatos de forma interativa.

Com a lista de contatos, os usuários podem adicionar novos contatos fornecendo nome e e-mail, remover contatos existentes e visualizar a lista completa de contatos cadastrados.
